# Vercel Logs

## List Deployments

```bash
vercel ls
```

## View Logs

```bash
vercel logs <deployment-url>
```

**Follow logs in real-time:**
```bash
vercel logs <deployment-url> --follow
```

## Analyze

- Look for errors or warnings
- Check for failed function invocations
- Identify build failures
